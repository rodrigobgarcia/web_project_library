Escolhas do Leitor

Livros esssenciais

  • Clean Code: Habilidades Práticas do Agile Software

    Quer saber como escrever um código robusto, independentemente da arquitetura por trás da linguagem de programação escolhida? Em seguida, considere a leitura do livro: Code Complete: Um Manual Prático de Construção de Software. Construção. Ele aborda de forma abrangente tudo relacionado à boa estrutura de código.

    Robert C. Martin

  • Redes de Computadores e a Internet: Uma Abordagem Top-Down

    Se você deseja entender a fundo como a Internet e as redes de computadores funcionam, desde a camada de aplicação (os programas que você usa) até a camada física, este é o livro.

    James F. Kurose e Keith W. Ross

  • O Programador Pragmático: De Aprendiz a Mestre

    Este livro vai além do código em si, focando na mentalidade e nas práticas que transformam um bom programador em um desenvolvedor excelente. Ele trata de tópicos como responsabilidade pessoal, arquitetura de software, automação, testes, refatoração, e como lidar com projetos e equipes de forma eficaz e pragmática. É um guia para a filosofia do desenvolvimento de software.

    Andrew Hunt & David Thomas

  • Padrões de Projetos: Soluções Reutilizáveis de Software Orientado a Objetos

    Este livro é fundamental para qualquer desenvolvedor que queira construir software complexo de forma organizada e sustentável. Ele cataloga 23 padrões de projeto de software cuja aplicação é crucial para escrever um código mais flexível, reutilizável e fácil de manter.

    Erich Gamma, Richard Helm, Ralph Johnson, John Vlissides (Gang of Four)
